# Change Log

All notable changes to the "command-output-hover" extension will be documented in this file.

## [0.0.1] - 2024-11-12

### Added
- Initial release
- Execute shell commands with selected text as input
- Display command output in hover tooltips
- Support for Bash and PowerShell
- Configurable command templates with `{{input}}` placeholder
- Configurable timeout for command execution
- Progress indicator during command execution
- Error handling and notifications
- Automatic cleanup of stale hover data
- Keyboard shortcut (Ctrl+Shift+E / Cmd+Shift+E)
